# Inventory Management Report (SAP ABAP)

## 📌 Project Overview

This project is an Inventory Management Report developed using SAP ABAP.  
It manages product inventory using internal tables and performs CRUD operations (Add, Update, Delete).

---

## ⚙️ Features

* Add new products to inventory
* Update product quantity
* Delete products from inventory
* Display inventory list
* Duplicate product check
* Error handling for missing products

---

## 🛠️ Tech Stack

* SAP ABAP (Eclipse ADT)
* Object-Oriented ABAP
* Internal Tables
* Open SQL concepts

---

## 📊 Output

Displays inventory data with:

* Product ID
* Name
* Quantity
* Price
